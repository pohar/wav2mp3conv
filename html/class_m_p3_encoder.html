Detailed Description

MP3Encoder class.

The MP3Encoder class converts the sample data in the incoming array(s) to and mp3 file, and saves it to the disk.

void MP3Encoder::Encode ( )

The function that actually does MP3 encoding.

MP3 encoding function. The size of encoding buffer is determined by Lame documentation. After initalizing the Lame library and the parameters are set, the encoding happens (depending on the audio format and the number of channels). If the encoding was successful, the mp3 file is written.
